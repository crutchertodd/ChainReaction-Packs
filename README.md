# ChainReaction Packs

Public level pack feed for the ChainReaction app.

## Structure

- `packs/catalog.json` - index consumed by the app
- `packs/*.json` - level pack files

## Catalog format

```json
{
  "packs": [
    {
      "packId": "zero-g",
      "version": 1,
      "filename": "zerog.json"
    }
  ]
}
```
